
Example:

1. vim Hello.java

2. press any of i,I,o,O,a,A,r,R to start editing

3. insert code
```java
public class Hello{
	public static void main(String[] args){
		System.out.println("Hello world");
	}
}
```

4.  press esc and type :wq, then press enter


---
some shortcuts:
:wq
save and exit

:q
quit without save

q!
force to quit without save
