### Zip vs Merge vs Concat
```java
Flux<Integer> a = Flux.just(1,2,3).delayElements(Duration.ofMillis(30));
Flux<Integer> b = Flux.just(10,20,30);

Flux.zip(a,b).subscribe(System.out::println);     // pairs in order of both
Flux.merge(a,b).subscribe(System.out::println);   // interleave by arrival time
Flux.concat(a,b).subscribe(System.out::println);  // a then b
```

### Windowing
```java
Flux.range(1, 20)
    .window(5)
    .flatMap(win -> win.reduce(0, Integer::sum))
    .subscribe(System.out::println); // sum per window
```
