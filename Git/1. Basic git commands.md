
### Initialization:

```
git init git
add . 
git commit -m “init”
```

### Config:

Setting User Name and Email:

```
git config --global user.name "Your Name"
git config --global user.email "youremail@example.com"
```

Setting Text Editor:

```
git config --global core.editor "nano"
```

Default Branch:

```
git config --global init.defaultBranch "main"
```
